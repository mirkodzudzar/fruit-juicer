# Fruit Juicer Simulation

## Requirements

- PHP 8.3
- Composer installed

## Setup and Run

1. Clone the repository:

   ```bash
   git clone https://github.com/mirkodzudzar/fruit-juicer
   cd fruit-juicer
   ```

2. Install dependencies (may be skipped):

   ```bash
   composer install
   ```

3. Run the script

   ```bash
   php scripts/run.php
   ```
